# 🌡️ 세션 웜업 (Warmup) 워크플로우

> **목표**: 새로운 세션 시작 시, AI가 프로젝트의 전체 구조와 핵심 맥락을 빠르게 파악하여 "기억 상실"을 방지함.

## 1. 디렉토리 구조 파악

1. 프로젝트 루트에서 `list_dir`을 실행하여 전체 폴더 구조를 파악한다.
2. `src/`, `docs/`, `scripts/` 등 주요 폴더의 하위 구조도 확인한다.

## 2. 필수 문서 정독

다음 파일들을 **반드시** 읽어야 한다 (`read_file`):

1. **`.agent/constitution.md`**: 헌법 (절대 규칙)
2. **`docs/ISSUES.md`**: 현재 진행 중인 작업과 남은 작업 (가장 중요)
3. **`docs/HANDOVER.md`**: 이전 세션에서 넘겨준 상세 인계 사항
4. **`docs/ARCHITECTURE.md`** (있다면): 시스템 아키텍처
5. **`package.json`**: 프로젝트 의존성 및 스크립트

## 3. 소스 코드 샘플링

- `src/` 폴더 내의 모든 파일을 열어서 코드 스타일과 구조를 파악한다.
- 예: `src/index.js`, `src/core/Manager.js` 등

## 4. 준비 완료 보고

- 위 과정이 끝나면 사용자에게 "준비 완료. 현재 상황 파악했습니다."라고 보고한다.
- **다음 작업 제안 금지** (사용자의 명령을 기다릴 것).
